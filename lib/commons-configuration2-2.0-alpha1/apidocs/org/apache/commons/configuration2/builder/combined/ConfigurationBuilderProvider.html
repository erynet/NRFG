org.apache.commons.configuration2.builder.combined

Interface ConfigurationBuilderProvider

  • All Known Implementing Classes:
    BaseConfigurationBuilderProvider, CombinedConfigurationBuilderProvider, FileExtensionConfigurationBuilderProvider, MultiFileConfigurationBuilderProvider

    public interface ConfigurationBuilderProvider

    An interface for creating builders for configuration sources of a CombinedConfigurationBuilder.

    When processing its definition file CombinedConfigurationBuilder scans for tags declaring configuration sources and maps them to implementations of this interface. The instances are then used to obtain builder objects to create the corresponding configuration sources. Parameters of the builders are provided as ConfigurationDeclaration objects.
