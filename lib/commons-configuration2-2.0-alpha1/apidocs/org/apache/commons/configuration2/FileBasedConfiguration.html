Interface FileBasedConfiguration

  • All Superinterfaces:

    All Known Implementing Classes:

    public interface FileBasedConfiguration
extends FileBased, Configuration

    An interface which combines the FileBased and Configuration interfaces.

    This interface is used to parameterize ConfigurationBuilder implementations which produce configurations read from a FileHandler.
